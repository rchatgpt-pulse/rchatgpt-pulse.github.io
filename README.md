Website source for https://rchatgpt-pulse.github.io/, a companion web app to the ICML 2026 paper [Three Years of r/ChatGPT: Societal Impact Evaluations from Social Media Data](https://arxiv.org/abs/2606.05750). 

To maintain compliance with Reddit API policy, daily scrapes of post texts (and their embeddings) are not available in this public repo; code for scraping & processing is included here for completeness.

Please contact jessicadai@berkeley.edu with questions or concerns.
